Compute Shader Experimentation

08 Apr 2017
  • Starting off from work done on Flood flood-optimization, got inspired to do more work on the GPU

Generate hex column meshes

  • Bundle many columns into one mesh.
  • 6 sides, a cap on top.
  • LOD0: Beveled edges on all sides. Added center vertex with triangles out from that as a cap.
  • LOD1: No bevels, triangle fan as cap.
  • All vertices except the bottom loop weight painted as 1.
  • Unity’s Meshes have a limit of 65535 vertices.
  • To work with Unity’s LOD system, both LODs contained the same number of columns. Even though the vertex limit would have allowed LOD1 to contain many more columns.
  • Maximum size of a ‘chunk’ was 31x31 using LOD0 meshes.

  • Prefer to stick to power of 2 numbers for things like this. Decided on 16x16 instead.

  • I could have absolutely created these meshes in something like Blender. I didn’t. Mesh generation is fun.
